# Helping Hands

A real-time AI assistant that provides contextual help during video calls, interviews, presentations, and meetings using screen capture and audio analysis.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- [npm](https://www.npmjs.com/)
- A [Google AI Studio](https://aistudio.google.com/apikey) API key

## Setup

1. Clone the repository:

```bash
git clone https://github.com/rajurathod0577/helping-hands.git
cd helping-hands
```

2. Install dependencies:

```bash
npm install
```

3. Start the app:

```bash
npm start
```

4. Enter your Gemini API key in the app and select a profile to begin.

## Keyboard Shortcuts

| Shortcut                | Action               |
| ----------------------- | -------------------- |
| `Ctrl/Cmd + Arrow Keys` | Move window          |
| `Ctrl/Cmd + M`          | Toggle click-through |
| `Ctrl/Cmd + \`          | Close / go back      |
| `Enter`                 | Send message to AI   |
